PurC - HVML 解释器

PurC 是魏永明领导开发团队开发的第一个 HVML 解释器，使用 C 编程语言。PurC 是 Purring Cat（呼噜猫）的缩写，而呼噜猫是 HVML 的吉祥物。

HVML 应用框架

在完整的基于 HVML 的应用框架中，包含一个独立运行的图形用户界面渲染引擎（简称“渲染器”），开发者通过撰写 HVML 程序来操控渲染引擎，而 HVML 程序在 HVML 解释器中运行，并可以和其他已有的编程语言运行时环境绑定起来，接收由其他编程语言程序生成的数据，并按照 HVML 程序的指令，将其转换为图形用户界面的描述信息或者变更信息。也就是说，在 HVML 的应用框架中，运行有两个主要组件。

HVML 解释器

HVML 解释器加载并解析 HVML 程序，然后以特有的方式运行 HVML 程序，并生成使用 HTML、XML 等标签语言描述的文档或文档的变化信息，然后交由渲染器渲染。解释器同时监听来自渲染器交互事件并进行处理。PurC 便是 HVML 解释器的一种实现。

HVML 渲染器

渲染器根据 HVML 解释器生成的 HTML、SVG、MathML 文档并根据 CSS 的定义将文档渲染为最终的图形界面，然后 HVML 解释器生成的文档变化信息，更新界面并实现和用户的交互。xGUI Pro 便是这样一种渲染器的实现。

xGUI Pro

xGUI Pro is a modern, cross-platform, and advanced HVML renderer which is based on tailored WebKit.
